# MyGodItsFull0fStars Tab

## How do I install these formulae?

`brew install mygoditsfull0fstars/tab/<formula>`

Or `brew tap mygoditsfull0fstars/tab` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "mygoditsfull0fstars/tab"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
